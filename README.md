# 🏡 House Price Prediction Project

A professional End-to-End Machine Learning project that predicts house prices based on features like area, location, age, and property type. It includes a modular Python backend and a Streamlit web interface.

## 🚀 Project Overview
- **Objective**: Predict house prices using Random Forest Regressor.
- **Accuracy**: ~97% R² Score.
- **Tech Stack**: Python, Scikit-Learn, Pandas, Streamlit, YAML for Config.
- **Architecture**: Modular structure (Production-ready).

## 📁 Repository Structure
- `app/`: Streamlit web application.
- `config/`: Configuration files (YAML).
- `data/`: Sample house price dataset.
- `models/`: Trained model (Pickle file).
- `src/`: Source code for preprocessing and training.
- `requirements.txt`: List of dependencies.
